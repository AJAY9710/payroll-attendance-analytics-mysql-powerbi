# Payroll & Attendance Analytics System

## Project Overview
An end-to-end Payroll and Attendance Analytics system built using MySQL and Power BI. The project simulates a real-world HR and payroll database and provides analytical dashboards for business decision-making.

## Tech Stack
- MySQL 8.x (Workbench)
- SQL (Joins, CTEs, Window Functions)
- Power BI
- GitHub

## Key Features
- Normalized relational database design
- Automated payroll calculation using stored procedures
- SQL views for simplified reporting
- Advanced SQL queries for analytics
- Interactive Power BI dashboards with KPIs and filters

## Database Objects
- Tables: employees, departments, attendance, leaves, salary_structure, payroll
- Views: vw_employee_payroll, vw_attendance_summary
- Stored Procedures: payroll generation, department salary cost

## Dashboards
- Employee attendance percentage
- Present vs absent analysis
- Monthly payroll by employee
- Department-wise salary distribution
- Total payroll KPI
